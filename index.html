Arduino Notes

ðŸ”¹ Introduction to Arduino

What is Arduino?
Arduino is an open-source electronics platform based on easy-to-use hardware (microcontrollers) and software (IDE). It enables beginners and professionals to build interactive projects.

Why Arduino?

  • Simple and beginner-friendly
  • Inexpensive compared to other microcontrollers
  • Large community support and open-source libraries

Common Arduino Boards

  • Arduino Uno (most popular, ATmega328P)
  • Arduino Mega (more pins, more memory)
  • Arduino Nano (small, compact)
  • Arduino Leonardo, Due, ESP32-based boards, etc.

Applications of Arduino

  • Robotics
  • IoT (Internet of Things)
  • Home automation
  • Sensor-based projects
  • Educational purposes

ðŸ”¹ Arduino Hardware Basics

  • Microcontroller: The brain of the Arduino board
  • Digital I/O pins: For input (buttons, sensors) and output (LEDs, motors)
  • Analog input pins: For reading varying sensor values (e.g., temperature, light)
  • Power pins: 3.3V, 5V, GND
  • USB port: For programming and power
  • Reset button: Restarts the program

ðŸ”¹ Arduino Software (IDE)

The Arduino IDE is used to write, compile, and upload code (called sketches) to the board. The language is based on C/C++.

Structure of a sketch

void setup() {
  // runs once at the beginning
}

void loop() {
  // runs continuously after setup
}

ðŸ”¹ Arduino Programming Basics

  1. Variables & Data Types: int, float, char, boolean, String
  2. Functions: pinMode(), digitalWrite(), digitalRead(), analogRead(), delay()
  3. Control Structures: if, else, for, while, switch-case
  4. Comments: // single-line, /* multi-line */

ðŸ”¹ Example Program: Blink LED

Pin 13 usually has a built-in LED on many Arduino boards.

int led = 13;  // Pin 13 has a built-in LED

void setup() {
  pinMode(led, OUTPUT);  // set pin as output
}

void loop() {
  digitalWrite(led, HIGH);  // turn LED on
  delay(1000);              // wait 1 second
  digitalWrite(led, LOW);   // turn LED off
  delay(1000);              // wait 1 second
}

ðŸ”¹ Tips for Beginners

  • Always check board type and port in Arduino IDE before uploading.
  • Use the Serial Monitor (Serial.begin(9600);) for debugging.
  • Start with simple projects: LED blink, button input, sensor reading.
  • Gradually move to advanced projects: IoT, robotics, automation.
